About this project

The goal of this project is to create a simple static build website with no dependencies and no run-times, available on all major platforms, using Golang with Echo Framework and Hero templates with Boostrap 4.

Read more at github.com/OviCrisan/SiteGo.

For a ASP.NET Core version see github.com/OviCrisan/SiteNet.

Please visit my website Crisan.ca or my blog Ovi.Crisan.ca or follow me on Twitter @Ovi_Crisan.
